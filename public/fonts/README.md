## Required Fonts

### 1. GFS Didot (Custom .otf)
Place your Didot font files here:
- `GFSDidot-Regular.otf`
- `GFSDidot-Bold.otf` (if available)

### 2. Sofia Sans
Will be loaded from Google Fonts.

## File Naming Convention

Use clear, consistent names:
```
GFSDidot-Regular.otf
GFSDidot-Bold.otf
SofiaSans-Regular.otf (if using local version)
SofiaSans-Bold.otf
```

## After Adding Fonts

1. Place your .otf files in this directory
2. The fonts will be automatically loaded via `globals.css`
3. Test with: `npm run dev`
